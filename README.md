# 🎮 LÊ HẢI LONG — Creative Quest

Portfolio cá nhân phong cách **8-bit arcade / synthwave** của Lê Hải Long (Video Editor & Creative — CR Studio).

Trải nghiệm như một game retro: **Loading → Start Game → Chọn nhân vật → Hồ sơ người chơi** (About, Showreel, Liên hệ).

## ✨ Tính năng

- Màn hình mở đầu kiểu arcade với nút **START GAME**
- **Chọn nhân vật** với video chân dung pixel-art tự phát
- **Player Profile**: BIO, thanh chỉ số kỹ năng (HP/MP), trang bị (Premiere/AE/DaVinci…)
- **Showreel**: lưới video YouTube **tự phát khi rê chuột**
- **Liên hệ**: Email, Facebook, mã QR Zalo
- Nền **synthwave grid** chuyển động + hiệu ứng **CRT scanline**
- Theme **"Hextech & Shadow"** (tím / cyan / vàng kim) — cảm hứng Arcane
- Font pixel hỗ trợ tiếng Việt: SVN-Determination · LanaPixel · SVN-Coder's Crux

## 🛠️ Công nghệ

- [Next.js 16](https://nextjs.org) (App Router)
- [Tailwind CSS 4](https://tailwindcss.com)
- TypeScript
- Canvas 2D (synthwave grid)

## 🚀 Chạy local

```bash
npm install
npm run dev
```

Mở [http://localhost:3000](http://localhost:3000).

## 📦 Build

```bash
npm run build
npm start
```
